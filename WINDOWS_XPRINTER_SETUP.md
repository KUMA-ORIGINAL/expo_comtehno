# Настройка авто-печати билетов на Windows (XPrinter)

Ниже пошаговая инструкция для стойки регистрации, где билеты печатаются сразу после сканирования.

## 1) Подготовка принтера

1. Подключите XPrinter к компьютеру.
2. Проверьте, что принтер печатает тестовую страницу в Windows.
3. Откройте:
   - `Параметры` -> `Bluetooth и устройства` -> `Принтеры и сканеры`
4. Выберите ваш XPrinter и установите его как принтер по умолчанию.

## 2) Проверка рабочей страницы

Перед настройкой авто-печати убедитесь, что открывается страница сканирования персонала:

- [http://localhost:8000/checkin/](http://localhost:8000/checkin/)

Если страница не открывается:
- проверьте, что Django-сервер запущен;
- проверьте правильность порта (`8000`).

## 3) Авто-печать через Chrome

Создайте файл `start-checkin-chrome.bat` (например, на Рабочем столе) с таким содержимым:

```bat
@echo off
start "" "C:\Program Files\Google\Chrome\Application\chrome.exe" --kiosk --kiosk-printing --new-window "http://localhost:8000/checkin/"
```

Если Chrome установлен в другом каталоге — укажите ваш путь к `chrome.exe`.

## 4) Авто-печать через Edge (альтернатива)

Создайте файл `start-checkin-edge.bat`:

```bat
@echo off
start "" "C:\Program Files (x86)\Microsoft\Edge\Application\msedge.exe" --kiosk --kiosk-printing --new-window "http://localhost:8000/checkin/"
```

Если путь к Edge отличается — замените его на актуальный путь к `msedge.exe`.

## 5) Как работает процесс на стойке

1. Персонал открывает `/checkin/`.
2. Сканирует QR билета.
3. Нажимает кнопку `Печать для XPrinter`.
4. Если браузер запущен с `--kiosk-printing`, печать уходит сразу на принтер по умолчанию (XPrinter), без выбора `Save as PDF`.

## 6) Полезные замечания

- Обычный запуск браузера **без** `--kiosk-printing` будет показывать стандартный диалог печати.
- Для стабильной работы на стойке рекомендуется:
  - отдельный Windows-пользователь для регистрации;
  - отключение лишних уведомлений;
  - автозапуск `.bat` после входа в систему.

## 7) Быстрая проверка после настройки

1. Запустите `.bat`.
2. Откройте билет и нажмите `Печать для XPrinter`.
3. Убедитесь, что чек выходит сразу на XPrinter.
